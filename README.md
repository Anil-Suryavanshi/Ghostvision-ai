# GhostVision AI

GhostVision AI is a simple tool that analyzes a GitHub repository and helps in understanding the code and fixing issues.

This project is built to make it easier for developers to explore code, identify problems, and get suggested fixes without going through every file manually.

---

## Features

- Analyzes the structure of a GitHub repository  
- Reads important code files  
- Generates fixes based on the given issue  
- Helps in creating pull requests  
- Simple interface for easy use  

---

## Tech Stack

- Node.js  
- Express.js  
- GitHub API  
- Gemini AI  
- HTML, CSS, JavaScript  

---

## How to Use

1. Enter the GitHub repository URL  
2. Write the issue or problem  
3. Click on "Fix with GhostVision"  
4. The system will analyze the repository and suggest a fix
